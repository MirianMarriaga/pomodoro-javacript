# Pomodoro App

Final project for the Codédex JavaScript course, based on their Pomodoro App tutorial:
[Link here!](https://www.codedex.io/projects/build-a-pomodoro-app-with-html-css-js)

### Features

- Pomodoro timer with 25, 5, and 15 minute sessions
- Task Mode with a task counter
- Automatic work/break cycles until all tasks are completed

### Built With

- HTML
- CSS
- JavaScript
